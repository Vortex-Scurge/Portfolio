Description

DevPilot is an advanced, automated static analysis and AI-code-review system. It seamlessly clones repositories, performs deep static analysis, and uses intelligent LLM integration to generate a comprehensive health, maintainability, and security report.

Features

  • AI-Powered Code Reviews using Groq LLM integration
  • Automated Security Scanning to discover vulnerabilities
  • Asynchronous Background Processing with Celery and Redis
  • Real-time Interactive Dashboard built in React
  • Fully Dockerized architecture for seamless deployment

What I Learned

  • Orchestrating complex asynchronous workflows using Celery and Redis
  • Integrating LLMs (Groq) for contextual code analysis and vulnerability remediation
  • Utilizing AST and static analysis tools (like Bandit) in an automated pipeline
  • Designing real-time polling mechanisms in React for background tasks
  • Containerizing a multi-service architecture (FastAPI, React, Postgres, Redis, Celery)

Future Improvements

  • Support for more programming languages and language-specific static analyzers
  • Integration directly into GitHub PR workflows via GitHub Apps/Webhooks
  • Fine-tuning a smaller, local LLM to reduce API costs for code reviews
  • Adding historical tracking to measure code maintainability over time
